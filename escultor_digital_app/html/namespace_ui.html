Descrição Detalhada

The definecolors class a classe responsável pela caixa de diálogo que é aberta quando o usuário clica no método Color onde o usuário pode definir a cor dos objetos que serão desenhados pelos os outros métodos.

The tamanhoSphere class a classe responsável pela caixa de diálogo que é aberta quando o usuário clica no método putSphere onde o usuário pode definir o raio da esfera.

The TamanhoSculptor class classe responsável pela caixa de diálogo aberta quando o usuário clica no mudardimensão onde ele pode setar o tamanho dos planos x,y,z.

The tamanhoEllipsoid class responsável pela caixa de diálogo aberta quando o usuário clica no putellipsoid e o usuário fornece os parâmetros do elipsoide as coordenadas x,y,z e o raio.

The tamanhoBox class a classe responsável pela caixa de diálogo que é aberta quando o usuário clica no método putBox onde o usuário pode definir o tamanho da caixa nos eixos x,y,z.

The MainWindow class a classe responsável pela tela principal do projeto. Nela há as conexões devido as ações que podem acontecer com a interação do usuário. Possui as funções responsáveis pela interação e mudança dos parâmetros fornecidos pelo usuário para o desenho das figuras geométricas.

getColorR() retorna o valor inteiro da cor no índice Red. getColorG() retorna o valor inteiro da cor no índice Green. getColorB() retorna o valor inteiro da cor no índice Blue.

d - declaração da caixa de diálogo responsável pelo tamanho do scultor. dialog_pb - declaração da caixa de diálogo responsável pelo tamanho do box. dialog_ps - declaração da caixa de diálogo responsável pelo tamanho da esfera. dialog_pe - declaração da caixa de diálogo responsável pelo tamanho do elipsóide. dialog_pc - declaração da caixa de diálogo responsável pela definição da cor.

getBoxX() - retorna o valor inteiro do tamanho no eixo x. getBoxY() - retorna o valor inteiro do tamanho no eixo y. getBoxZ() - retorna o valor inteiro do tamanho no eixo z.

getRaioX - pega o raio do elipsóide no eixo x. getRaioY - pega o raio do elipsóide no eixo y. getRaioZ - pega o raio do elipsóide no eixo z. getRaioE - tamanho do raio geral da elipsóide.

getX() - pega o valor inteiro do plano x fornecido pelo usuário. getY() - pega o valor inteiro do plano y fornecido pelo usuário. getZ() - pega o valor inteiro do plano z fornecido pelo usuário.

função getRaio() retorna o valor inteiro do raio.
